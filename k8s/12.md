# Implementing visits

Python app:

```python
@app.route('/visits')
def read_visits():
    try:
        with open('./volume/visits') as f:
            visits = int(f.read())
    except FileNotFoundError:
        visits = 0

    return str(visits)
```


Typescript app:

```typescript

            setVisits(data.visits);
```

### Env Config mapping

`configmap.yaml`

```yaml
apiVersion: v1
kind: ConfigMap
metadata:
  name: config-configmap
data:
  config.json: |-
{{ .Files.Get "files/config.json" | indent 4}}
```


`env-configmap.yaml`:

```yaml
apiVersion: v1
kind: ConfigMap
metadata:
  name: env-configmap
data:
  SECRET_LAB: "SECRET_LAB"
```
